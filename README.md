# Week 6 Side Quest

Liza Mehta, l3mehta, 21080966

A 2D side-scrolling platformer built with [p5.js](https://p5js.org/) and [p5play](https://p5play.org/) for GBDA302.

## Controls

| Key       | Action                       |
| --------- | ---------------------------- |
| `A` / `←` | Move left                    |
| `D` / `→` | Move right                   |
| `W` / `↑` | Jump                         |
| `Space`   | Attack                       |
| `R`       | Restart (after death or win) |

## How to Play

Collect all 15 leaves to win. Avoid snakes and fire — each hit costs a life. You have 3 lives. Attack snakes by pressing Space while facing them on the ground.
